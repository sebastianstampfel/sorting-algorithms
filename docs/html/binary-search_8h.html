Detailed Description

Adds functionality for task 2 - binary search.

Generates a randomized list of nodes and prompts the user to input a number to search for. Prints found number.

void binarysearch ( int  amount)

Function for the whole binary search operation.

Generates a list of n nodes (n = amount), and fills the nodes with random data. Sorts the list in ascending order using qsort() afterwards. User will be prompted to enter a number to search for. If the search criteria is met by one element, this element will be printed to stdout.
